Around The U.S.

Purpose & Concept

“Around The U.S.” is a responsive web app inspired by social media platforms, where users can edit their profile, update their avatar, and share locations with images and captions. Built with HTML5, CSS3, and modern JavaScript, it emphasizes clean design, usability, and dynamic interaction.

Key Features

  • Editable profile with avatar updates
  • Interactive cards for places with likes and delete options
  • Add new locations dynamically through forms
  • Pop-up modals with smooth transitions
  • Responsive layout using flexbox and grid

What It Demonstrates

This project highlights my ability to design and implement interactive front-end applications, apply real-time validation, and manage dynamic DOM updates. It showcases critical thinking under pressure and adaptability across devices.
